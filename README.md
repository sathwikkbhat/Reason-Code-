# ReasonCode — Coming Soon Landing Page 🚀

"Learn the reasoning, not just the solution. Be the first to know when we launch."

This repository contains the "Coming Soon" waitlist page for **ReasonCode**, built with Astro.

## Features
- **Modern & Premium Design:** Sleek developer theme with coding font styling, grid overlays, and neon green highlights.
- **Form Ingestion:** Integrated with Mailchimp waitlist subscription.
- **Security Protections:** Form honeypot and secure link attributes.
- **Fully Responsive:** Layout renders perfectly across mobile, tablet, and desktop views.

## Getting Started

### Local Development
To run the server locally:
```sh
npm install
npm run dev
```

The site will be available at `http://localhost:4321/`.

### Production Build
To compile the static site output:
```sh
npm run build
```

The compiled output will be exported to the `dist/` directory.
